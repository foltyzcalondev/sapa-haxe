# sapa-haxe
Sebuah library/pustaka sederhana untuk menyapa pengguna di terminal, ditulis menggunakan bahasa Haxe.

## Instalasi:
```{bash}
haxelib install sapa
```
## Contoh penggunaan:
```(haxe}
// Main.hx

import sapa.User;

class Main {
  static function main() {
    trace(User.sapa("Budi", 30, "Jakarta");
  }
}
```
